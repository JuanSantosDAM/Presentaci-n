Proyecto: Identidad & Visión 🚀
Bienvenido al repositorio de mi portafolio personal. Este proyecto es una tarjeta de presentación digital diseñada para mostrar quién soy, mi filosofía de trabajo y los proyectos en los que he estado sumergido durante mi formación.

Juan Marcos Santos Cotelo Estudiante de 1º de DAM (Desarrollo de Aplicaciones Multiplataforma).

📝 Descripción de la Práctica
Esta práctica consiste en el desarrollo de una landing page responsiva que utiliza una estructura de cuadrícula moderna. El objetivo principal es aplicar conceptos avanzados de maquetación web para organizar información personal y profesional de forma visualmente atractiva y funcional.

Secciones incluidas:

Header: Título dinámico con línea divisoria minimalista.

Sobre Mí: Breve introducción sobre mi pasión por el aprendizaje y la programación.

Intereses: Un desglose visual de mis hobbies fuera del entorno de código.

Filosofía: Una sección destacada con mi visión sobre lo que significa programar.

Laboratorio (Footer): Un espacio dedicado a mis proyectos académicos y personales, conectado directamente con GitHub.

🛠️ Tecnologías Utilizadas
Para la construcción de este sitio he empleado las siguientes herramientas y técnicas:

Estructura (HTML5)
Etiquetas Semánticas: Uso de header, main, section, article y footer para una mejor accesibilidad y SEO.

Navegación: Enlaces externos configurados para abrirse en pestañas nuevas (_blank).

Estilos y Maquetación (CSS3)
CSS Grid: Utilizado para la distribución principal del contenido, incluyendo el manejo de filas expandidas (grid-row).

Flexbox: Aplicado en el header y footer para el alineamiento preciso de elementos internos.

Variables CSS (:root): Definición de una paleta de colores coherente y tipografías reutilizables.

CSS Nesting: Implementación de anidamiento nativo para un código más limpio y legible.

Diseño Responsivo: Uso de Media Queries para asegurar que la web se vea perfecta tanto en monitores de escritorio como en dispositivos móviles.

Interactividad: Efectos de transición (transition) y transformaciones (transform: translateY) en el hover de las tarjetas.
